# 🦎 Wildlife Tracker
A desktop app for managing and tracking animals, built with C# and WPF as part of a university course assignment. It's a 
functional tool that puts object-oriented programming principles to work together with a UI.

## What it does
- Add, edit and remove animals and their details
- Manage a feeding schedule for every animal
- Sort animals by name or species using custom comparers
- Use serialization to store and load data in different ways using text files, and serialization
in Json and XML formats.

## Built with
- C#
- .NET
- WPF (Windows Presentation Foundation)

## Project purpose
This started as a C# course assignment, but it turned into a decent showcase of desktop development and OOP design.

## How to Run
Requirements: Windows OS, Visual Studio (2022)
Make sure the **.NET desktop development** workload is selected during installation.

Clone the repo:
bashgit clone https://github.com/nikolinnav/wpf-wildlife-tracker.git

Open the .sln file in Visual Studio
Build and run.
